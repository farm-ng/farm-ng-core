65 // Add a "3-vec curve" to the plot or append to exiting one with path being
66 // "plot_name/curve_name". If a curve with that name already exists, it will be
67 // overwritten.
68 //
69 // A "3-vec curve" are 3 curves described a list of (3+1)-dimensional tuples
70 // `x_vec_pairs`, where the first component of the tuples is the x coordinate
71 // for each curve, and the remaining components are the y coordinates.
