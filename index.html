Core layer

Foundational core layer for new c++ projects. farm-ng-core only has a small set of dependencies: libfmt, expected (both being targeted for c++ standardization) and Protocol Buffers / GRPC.

Convenient macros

FARM_ENUM enables to/from string conversions. FARM_ASSERT_* for documenting and enforcing preconditions. FARM_INFO_* log macros with run-time and compile time log levels. Convient macros around Expected and more.

Event logging and more

EventLogReader and EventLogWriter to create and playback Protocol Buffer-based datasets and some more general purpose c++ utilities.
